# NLW Expert

Uma aplicação de notas feita em React

__Principais Features:__
- Criação de uma nota através de texto
- Gravação de uma nota através de áudio
- Salva as notas no localStorage do browser

  __Tecnologias:__
  - React + Vite
  - Radix
  - TailwindCSS
  - Sonner
  - SpeechRecognitionAPI
